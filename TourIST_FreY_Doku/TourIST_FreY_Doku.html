TOURIST°FREY – AR-App Starterpaket

Cross-Platform AR-App – Unity (iOS & Android), GPS/Location-basierte Marker, Objekterkennung, OCR, Barcode, Face Detection. Datenschutz/Consent, Demo-Marker, modernes UI-Panel-System.
Diese Seite ist als Doku und Quickstart für Entwickler/Freelancer/Teams gedacht.

1. Projektüberblick

  • Unity 2021.3+ / AR Foundation – voll plattformübergreifend
  • GPS/Kompass, Location-basierte Marker (Dummy- und Live-Events)
  • Modulare Features: Objekterkennung, OCR, Barcode, Face Detection
  • GDPR/DSGVO-Ready (Consent, Privacy, Datenlöschung)
  • Modernes UI mit versteckten Panels (randbasiert, swipefähig)
  • SurrealDB als Datenbank (REST/WebSocket)

2. Architektur-Übersicht

3. Feature-Liste

  • Versteckte, randbasierte Panels (Menü, Filter, Info)
  • Touch/Swipe-fähig, responsive UI
  • GPS- und Kompassintegration
  • Marker mit Label/Distanzanzeige (Dummy und dynamisch)
  • Integration für SurrealDB (REST/WebSocket)
  • Plug-ins für Objekterkennung (OpenCV), OCR (Tesseract), Barcode (ZXing), Face (Barracuda)
  • Consent-Dialog, In-App Privacy Policy, Impressum, Datenlöschung

5. Quickstart-Guide

  • Voraussetzungen: Unity 2021.3+ / AR Foundation / ARCore/ARKit XR Plugin
  • Projekt/Unitypackage importieren
  • AR Foundation & Plugins installieren (Package Manager)
  • Szene MainARScene.unity öffnen
  • DummyMarker platzieren, Panels & ConsentPanel prüfen
  • Build auf Android/iOS testen

6. Code-Beispiel: GPSManager

public Vector2 GetCurrentGPS()
{
    var loc = locationProvider.CurrentLocation;
    return new Vector2((float)loc.Latitude, (float)loc.Longitude);
}

7. GDPR & Datenschutz

  • Consent-Dialog beim Start (ConsentManager.cs)
  • Nur Meta-/Erkennungsdaten gespeichert, keine Bilder
  • Lokale Verarbeitung (OCR, Objekterkennung, Face)
  • TLS/HTTPS für SurrealDB-Verbindung
  • In-App Löschung, Opt-out, Privacy Policy/Impressum

8. UI/UX Panels

  • Links: Menü (Karte, Einstellungen, Impressum)
  • Rechts: Feature-Toggles (Objekt, OCR, Barcode, Face)
  • Oben: Adress-/Statuszeile
  • Unten: Info/Status

9. Marker & Overlay

  • AR-Marker mit GPS/Kompass platziert (Dummy oder dynamisch aus SurrealDB)
  • Overlay zeigt Name, Distanz (MarkerOverlayUI.cs)
  • Mehrere DummyMarker als Demo im Projekt
